### 3. 主板焊接

主板原理图：![](./img/ylt.png)

1. 将DC-DC 升压模块上的 3296W电位器取下
2. 将拨盘齿轮电位器(R2) 和 电阻(R3) 并联
3. 然后再将齿轮电位器和 3296W电位器(R1) 串联
4. 再将新的电位器焊接上 DC-DC升压模块（注意！！！一定要焊接上才能通电测试，否则有可能烧坏升压芯片）
5. 电位器电阻越大，输出的电压越高，一般最高电压调到11V左右。
